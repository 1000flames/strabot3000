# TenderRadar

A light, pitch-style landing page for a tender-document scraping product.

## What it shows

- A polished sales page
- A live feed of scraped tender documents
- Proof that the scraper already works

## Preview

Open `index.html` directly in a browser, or run:

```bash
python3 -m http.server 8000
```

Then visit `http://localhost:8000`.

## Demo data

The live feed is rendered from `data/autobahn_tenders.json`, which is generated by the scraper script in `scripts/autobahn_scraper.py`.
